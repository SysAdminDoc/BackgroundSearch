# BackgroundSearch

Chrome extension that forces all new tabs to open in the background without stealing focus.

## Features

- **Force Background Tabs** — New tabs open behind your current tab instead of switching to them
- **One-Click Toggle** — Click the toolbar icon to enable/disable (blue = on, gray = off)
- **Zero Config** — No popups, no settings pages, just works

## Install

1. Go to `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked** and select this folder
